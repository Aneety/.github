# Aneety

Aneety é uma plataforma white-label para pedidos customizados, produção ou execução, evidências, logística, pagamentos, marketplace operacional e administração por tenant.

## Repositórios principais

- [`Aneety/ai`](https://github.com/Aneety/ai) — orquestrador limpo da nova plataforma.
- [`Aneety/.github`](https://github.com/Aneety/.github) — documentação canônica da organização.
- [`Aneety/assets`](https://github.com/Aneety/assets) — assets reutilizáveis de marca.

## Documentação

A documentação oficial fica em [`docs/`](https://github.com/Aneety/.github/tree/main/docs).

Os repositórios históricos Lia permanecem arquivados na organização Malnati como fonte de aprendizado e evidência, não como base de implementação contínua.
